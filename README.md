# 🔥 Habit Tracker (React Native + Expo)

A clean and modern habit tracking app built with React Native and Expo. This app allows users to create habits, mark them as complete, and track progress using a dynamic streak counter.

Designed with a rich, earthy color palette for a premium and calming user experience.

---

## 🚀 Features

- 🔐 Login screen (UI-based)
- 🏠 Home dashboard
- ➕ Add new habits
- ✅ Mark habits as complete
- 🔥 Dynamic streak counter (increments on each completion)
- 🎨 Custom UI with curated color palette
- 🔄 Navigation using React Navigation
- 🧠 Global state management using Context API

---

## 📱 Screens

### 1. Login Screen
- Email & password input
- Navigates to Home screen

### 2. Home Screen
- Entry point to Habit Tracker

### 3. Habit Screen
- Add new habits
- View all habits
- Mark habits as complete
- Track streak count
